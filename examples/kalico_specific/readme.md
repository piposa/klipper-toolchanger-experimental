# Kalico
In here youll find macros specific to some of kalicos features which adapt things for printers with a toolchanger

## mpc_control

### SET_MPC_MATERIAL
A macro that generates the required slicer calls to add to your print start.
Configures your extruders for the right material at the start of a print.

### MPC_CALIBRATE
just a wrapper, makes calibration less typing
